Some details:
 1.1)  Intel RAO-INT instructions.
       RAO (Register atomical operations?) for integer.
       Atomically ADD, AND, OR, XOR instructions for destination=memory, source=register.
       Optimize atomical read-modify-write operations with data arrays, accessed by >1 execution thread.
 1.2)  Intel CMPccXADD instructions.
       Compare and add if condition (cc) is met.
 1.3)  Intel ArchPerfMonExt (Leaf 23).
       Architectural Performance Monitoring Extended Leaf (CPUID function 23h).
 1.4)  Intel WRMSRNS instruction.
       Non-serializing write to model-specific register.
 1.5)  Intel AMX-FP16 operations.
       Hardware matrix multiply with FP16 elements, TDPFP16PS instruction.
 1.6)  Intel AVX-IFMA instructions.
       128/256-bit version for IFMA instructions, VPMADD52HUQ, VPMADD52LUQ instructions.
 1.7)  Intel MSRLIST, means RDMSRLIST, WRMSRLIST instructions.
       Read and Write list of model-specific registers.
 1.8)  Intel AVX-VNNI-INT8 instructions.
       128/256-bit version for VNNI instructions with 8-bit integers.
 1.9)  Intel AVX-NE-CONVERT instructions.
       128/256-bit version for convert between BF16, FP16, FP32.
 1.10) Intel PREFETCHIT0/1 instructions.
       Prefetch hints for instruction cache.
 1.11) Intel undocumented function, only registers hex dump show.
